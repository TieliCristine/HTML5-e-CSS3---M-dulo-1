Inserindo vídeos do YouTube

Vantagens:

  • É gratis, o que é ótimo rsrs
  • Não consome banda do site pois é reproduzido de outro site
  • Podemos subir vídeos gratuitamente

Desvantagens:

  • Todos podem acessar o vídeo. No caso de produção de vídeo próprio não é muito legal, um exemplo é cursos online onde o vídeo só pode ser visto para quem é assinante.
  • Não consome banda do site pois é reproduzido de outro site
  • Aparece ao final do vídeo uma lista para ver outros vídeos

Inserindo vídeos do Vimeo

Vantagens:

  • Tem vídeos gratis, o que é ótimo também rsrs
  • Não consome banda do site pois é reproduzido de outro site
  • Vídeos de melhor qualidade
  • Não aparece no final do vídeo uma lista de sugestão para assistir outros vídeos

Desvantagens:

  • Par subir vídeos precisa ter uma assinatura
  • Há um limite de upload de vídeos
  • Uma assinatura é cara
