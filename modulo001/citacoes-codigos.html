Citações e Códigos

Códigos

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Para melhor leitura do código, utilizamos a tag "code": document.getElementById('teste') isso faz um leve espaçamento entre as letras e símbolos.

Para demonstrar um trecho de código em Python (ou outra linguagem), não basta apenas usar a tag "code", pois ficaria assim:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa.')

Para que a endentação seja considerada na apresentação do texto, precisamos envelopar a tag "code" com todo seu conteúdo, então ficará assim:

    
        num = int(input('Digite um número'))
        if num % 2 == 0:
            print(f'O número {num} é PAR')
        else:
            print(f'O número {num} é ÍMPAR')
        print('Fim do programa.')

Removendo o espaço o texto fica colado na esquerda:

    
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa.')
